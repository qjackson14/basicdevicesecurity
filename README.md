<h2>Step-by-Step</h2>

- 1. Change hostnames of Router and Switch using "hostname" command
 
  2. Configure an unencrypted enable password of 'CCNA' using "enable password" command
 
  3. Test password
 
  4. View password using "show running-config" command
 
  5. Ensure all passwords are encrypted using "service password-encryption" command
 
  6. View password using "show running-config" command
 
  7. Configure a more secure password 'Cisco' using "enable secret" command
 
  8. Save the running configuration to startup configuration using "write" command
 
  9. View startup configuration using "show start-up config" command
